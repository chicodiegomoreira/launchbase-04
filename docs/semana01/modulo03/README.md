### Instalação e Execução

Certifique-se de estar no mesmo diretório antes de executar estes comandos.

- Instalação:

```
    npm install
```

- Execução:
```
    npm start
```
Se tudo der certo, esta mensagem deve ser mostrada no console:

```
Server is running on port: 5000!
```
